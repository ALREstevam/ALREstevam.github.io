Heuristic formula explanation

f(n) = (a * pathDist(n)) + (b * goalDist(n)) + (c * difDist(n))
  • n: a node
  • f(n): the heuristic function
  • pathDist(n): the real distance in the path to the node i
  • goalDist(n): the distance between the start and the goal
  • difDist(n): the difficulty of the current path
